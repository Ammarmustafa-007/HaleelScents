# Haleel Scents - Premium E-Commerce Platform

A luxurious, high-performance e-commerce frontend built for **Haleel Scents**, a boutique fragrance brand. This project was developed with a focus on **rapid delivery**, **client acquisition**, and delivering an unforgettable digital experience through premium aesthetics and micro-animations.

## 🚀 Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Animations:** Framer Motion, Lottie React
- **Icons & UI:** Lucide React, React Icons
- **Routing:** React Router DOM (with fully animated page transitions)
- **Backend/Database:** Supabase & PostgreSQL (Ready for seamless auth and data management)
- **Build Tool:** Vite

## ✨ Key Features

- **Luxury UI/UX:** Dark-themed, glassmorphic design system tailored for high-end boutique brands.
- **Dynamic Routing & Animations:** Fluid page transitions using Framer Motion to keep users engaged without harsh page loads.
- **Custom Shopping Cart:** Global state management for adding items, tracking quantities, and calculating totals instantly.
- **Premium Product Showcases:** Immersive "quick-view" overlays utilizing React Portals to present product details (accords, notes, occasion) flawlessly on all screen sizes.
- **Mobile-First Excellence:** A custom-built, full-screen mobile drawer navigation and edge-to-edge product cards that look stunning on phones.
- **Rapid Client Delivery:** Built from the ground up for extreme speed-to-market while maintaining a world-class standard of quality.

## 📦 Setup & Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/haleel-scents.git
   ```
2. Navigate into the directory:
   ```bash
   cd haleel-scents
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠 Database & Backend (Supabase)
This project is configured to integrate with **Supabase** out of the box. 
- **PostgreSQL** serves as the robust relational database for products, orders, and user data.
- Built for scalable client acquisition, ensuring that as traffic scales, the backend infrastructure scales effortlessly.
